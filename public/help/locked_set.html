Locked Set

A Locked Set is a group of N cells in a single house with candidates for N digits.

The simplest locked set is a single candidate in a single cell, or a Naked Single.

Two candidates in two cells is called Naked Pair. Three candidates in three cells is called Naked Triple. Four candidates in four cells is called Naked Quad. In all cases, you can remove the candidates from other cells in the house.
